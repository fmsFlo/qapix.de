# Qapix - All-in-One Plattform für Finanzberater

Moderne B2B SaaS Landing Page für Qapix, inspiriert von Close.com mit Dark Theme, Glassmorphism und subtilen Animationen.

## Tech Stack

- **Next.js 14** mit App Router
- **TypeScript**
- **Tailwind CSS** für Styling
- **Framer Motion** für Animationen
- **Lucide React** für Icons
- **React Tilt** für 3D-Effekte

## Features

- ✅ Vollständig responsive Design
- ✅ Dark Theme mit Glassmorphism-Effekten
- ✅ Scroll-triggered Animationen
- ✅ SEO-optimiert mit Meta Tags
- ✅ Accessibility-freundlich
- ✅ Production-ready

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Build

```bash
npm run build
npm start
```

## Deployment

Die App ist bereit für Deployment auf Vercel:

```bash
vercel
```

## Struktur

```
/app
  /login          - Login Seite
  /impressum      - Impressum
  /datenschutz    - Datenschutz
  layout.tsx      - Root Layout mit Navigation & Footer
  page.tsx        - Landing Page
/components
  /sections       - Alle Landing Page Sections
  Navigation.tsx  - Sticky Navigation
  Footer.tsx      - Footer Component
```

## Design System

- **Primary Color**: #00D9FF (Cyan)
- **Secondary Color**: #7B61FF (Lila)
- **Dark Background**: #0A0E27
- **Fonts**: Space Grotesk (Headings), Inter (Body)
